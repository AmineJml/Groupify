# Groupify

Project by Abbas Harb

## Desciption
Groupify is a web app created to connect students from all majors together, where they can post problems or achievments they got in a school topic and share it with their<br>
fellow students.

## Stacks

1. HTML
2. CSS
3. Javascript
4. Axios
5. Laravel
6. JWT
